| Блок | Действие | Метод и путь | Описание |
|------|----------|--------------|----------|
| Auth | Регистрация | `POST /auth/register` | Создаёт пользователя по email + паролю |
| Auth | Логин | `POST /auth/login` | Аутентифицирует пользователя |
| Auth | Текущий пользователь | `GET /auth/me` | Возвращает текущего пользователя по токену |
| Пользователи | Поиск по нику | `GET /users?username=...` | Ищет пользователей по нику(логину) для старта чата |
| Пользователи | Просмотр своего профиля | `GET /profile` | Возвращает профиль текущего пользователя |
| Пользователи | Обновление профиля | `PUT /profile` | Обновляет профиль текущего пользователя |
| Пользователи | Просмотр чужого профиля | `GET /users/{userId}/profile` | Возвращает публичный профиль другого пользователя |
| Чаты | Список чатов | `GET /chats` | Список чатов пользователя (личные и групповые) |
| Чаты | Создать личный чат | `POST /chats` | Создаёт личный чат с другим пользователем |
| Чаты | Создать групповой чат | `POST /chats/group` | Создаёт групповой чат с переданным списком участников |
| Чаты | Информация о чате | `GET /chats/{chatId}/info` | Информация о конкретном чате |
| Чаты | Список участников | `GET /chats/{chatId}/members` | Список участников и их количество |
| Сообщения | История сообщений | `GET /chats/{chatId}` | История сообщений в чате |
| Сообщения | Отправить сообщение | `POST /chats/{chatId}` | Отправка сообщения  |
| Каналы | Список каналов | `GET /channels` | Список/поиск публичных каналов |
| Каналы | Информация о канале, подписаться/отписаться | `GET /channels/{channelId}/info` | Информация о канале (name, avatar, description) |
| Каналы | История сообщений | `GET /channels/{channelId}` | История сообщений канала |
| Каналы | Отправить сообщение | `POST /channels/{channelId}/messages` | Отправка сообщения в канал(owner) |
