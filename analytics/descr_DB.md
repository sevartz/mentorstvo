### User

- id: uuid v7 — идентификатор пользователя
- nickname: string — отображаемое имя, используется для поиска
- email: string (unique) — логин/идентификатор через oauth(вытаскивать)
- created_at: datetime — когда учетка создана

### Message

- id: uuid v7 — идентификатор сообщения
- sender_id: uuid v7 — ссылка на пользователя, отправителя
- receiver_id: uuid v7 — ссылка на пользователя, получателя
- text: text — тело сообщения в открытом виде (шифрование будет добавлено в MVP2).
- created_at: datetime — время отправки сообщения.
