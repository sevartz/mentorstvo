# mentorstvo
- **Идея**: минималистичный REALTIME месснджер с шифрованием, возможно с каналами.
импортозамещаем телегу, желательно сделать API под iosклиента

- **Язык / платформа:** C#, .NET 10
- **Web‑фреймворк:** ASP.NET Core
- **Real‑time:** SignalR (хабы для чатов)
- **Аутентификация:** Oauth2.0 и JWT токены (access/refresh токены)
- **БД:** PostgreSQL (пользователи, комнаты, сообщения, связи)
- **Очереди (на будущее, по требованиям):** RabbitMQ / Kafka (уведомления, фоновые задачи)
- **Log/Observability:** структурированные логи (Serilog), базовые метрики

### Инфраструктура

- **Контейнеризация:** Docker + docker‑compose (API, БД, Redis)
- **Миграции:** EF Core Migrations
- **Тесты:** xUnit / NUnit (юнит +, по возможности, интеграционные)

### Клиенты

1. **Простой веб‑клиент** (для начала, может быть минимальный):
   - JS/TS + любой удобный фронт (даже без фреймворка, либо React/Blazor).
   - SSE/SignalR‑клиент для проверки real‑time.
